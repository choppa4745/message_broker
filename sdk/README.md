# pigeonMQ SDK (Java)

Java client library for the pigeonMQ message broker.

Status: scaffold (module created). Public API will be added in the next commit.

## Modules covered
- MQTT publish/subscribe (topics + queues via shared subscriptions)
- Reconnect policies (fixed / exponential backoff)
- REST admin API client (create/delete topics and queues, health)

## Coordinates (planned)
```xml
<dependency>
    <groupId>com.pigeonmq</groupId>
    <artifactId>pigeonmq-sdk</artifactId>
    <version>0.1.0-SNAPSHOT</version>
</dependency>
```
